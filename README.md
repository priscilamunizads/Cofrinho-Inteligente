Sistema de Cofrinho Multimoedas

Este projeto é uma aplicação prática desenvolvida em Java que simula o funcionamento de um cofrinho digital interativo via terminal. O sistema permite armazenar moedas de diferentes nacionalidades e fazer o cálculo automático do valor total convertido para a moeda local.

Objetivo do Projeto

O foco principal foi colocar em prática os pilares da Programação Orientada a Objetos (POO), criando um menu interativo onde o usuário pode gerenciar suas economias de forma dinâmica e organizada.
